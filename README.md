# 🧠 Dopamine Detox AI

**AI‑powered digital wellness platform** – monitor your digital habits, get real‑time distraction alerts, and reclaim your focus through deep work sessions and behavioural insights.

---

## 📌 Project Overview

With increasing smartphone penetration, attention spans are shrinking. Dopamine‑driven platforms (Instagram, YouTube, TikTok) are designed to maximise engagement, leading to:

- Endless scrolling without time awareness
- High screen time during study/work hours
- Sleep disruption due to late‑night usage
- Constant notification interruptions
- Addiction‑like behaviour

**Dopamine Detox AI** provides an AI‑driven solution to monitor and manage digital habits, helping users regain control over their attention and time.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| **Beautiful login** | Any email/password works – auto‑registration |
| **Live distraction index** | Tracks screen time, interruptions, dopamine state (0–100) |
| **Deep work focus mode** | 10‑minute session that blocks simulated distractions and lowers your score |
| **App‑by‑app breakdown** | See time spent on Instagram, YouTube, TikTok, WhatsApp, Twitter, Gaming |
| **AI‑driven insights** | Personalised recommendations based on your usage pattern |
| **Real‑time alerts** | WebSocket‑powered background distractions (simulated, but realistic) |
| **Dark / light mode** | Toggle anywhere |
| **Distraction history log** | Last 20 events with timestamps |
| **About page** | Full problem statement and solution architecture |

---

## 🛠️ Tech Stack

- **Backend**: Node.js + Express + SQLite + Socket.io
- **Frontend**: HTML5, CSS3 (CSS variables, responsive), JavaScript (ES6+)
- **Authentication**: JWT (auto‑generated for any user)
- **Charts**: Chart.js
- **Real‑time communication**: WebSockets (Socket.io)

---

## 📦 Installation & Running Locally

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm (comes with Node.js)
